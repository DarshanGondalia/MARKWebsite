Project Description

The project is made for solving the difficulties of the Owner/Manager finding it difficult to manage the enormous database which he has to go through for every transaction thus making his job simple, easier and more convenient to work with.

This project will also aim at bill generation for customers. The system has to maintain the records of existing as well as previously available customers, suppliers and also the stock of items currently available.

The bill generation will be done by allotting each item a unique name or serial number. Shopping cart also called as electronic commerce system for customers to buy products using internet.
